# Cross-Platform Enigma Machine

A cross-platform Enigma machine simulator showcasing **Silicon-Logic Parity** — the same C++ codebase running on **Arduino**, **Desktop**, and **WebAssembly**.

## What This Demonstrates

This project proves **true cross-platform capability**:

- **Arduino UNO**: 2KB RAM, 16MHz CPU
- **Desktop**: Native C++ performance  
- **WebAssembly**: Runs in browser (Emscripten)
- **Same Code**: Zero modifications across platforms

> this project was made with [Nodepp](https://github.com/NodeppOfficial/nodepp) — a deterministic C++ runtime achieving zero memory leaks (Valgrind-verified).
